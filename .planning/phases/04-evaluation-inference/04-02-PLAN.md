  <name>Task 1: Create Predictor class with predict() and predict_batch()</name>
  <files>src/model/predictor.py, tests/test_predictor.py</files>
  <behavior>
    - Predictor.__init__ loads model from checkpoint, sets model.eval(), stores tokenizer and device
    - predict("Gemaess § 25a Abs. 1 KWG") returns list[PredictedSpan] with start, end, text, confidence fields
    - PredictedSpan.start and .end are character offsets into the original text
    - PredictedSpan.text == original_text[start:end]
    - PredictedSpan.confidence is in [0.0, 1.0] (mean softmax prob of B-REF/I-REF tokens in span)
    - predict on text with no references returns empty list
    - predict_batch(["text1", "text2"]) returns list[list[PredictedSpan]], one list per input
    - CRF mode: predict returns spans with confidence=1.0 (Viterbi does not produce marginals)
    - Inference uses torch.no_grad() context
    - predict completes in under 5 seconds (excluding model load time)
  </behavior>
  <action>
    **Create src/model/predictor.py** with:

    1. `PredictedSpan` dataclass: `start: int, end: int, text: str, confidence: float`

    2. `Predictor` class:
       - `__init__(self, checkpoint_path: str | Path, config, device: torch.device)` — create RegulatoryNERModel from config, call load_checkpoint, set model.eval(), move to device, create BertTokenizerFast from config.model.name. Store max_length from config.data.max_seq_length.
       - `predict(self, text: str) -> list[PredictedSpan]` — tokenize text with return_offsets_mapping=True, truncation=True, padding="max_length". Move tensors to device. Run model forward (no labels) under torch.no_grad(). Handle CRF vs non-CRF:
         - Non-CRF: logits = output.logits, pred_ids = logits.argmax(dim=-1), compute softmax probs
         - CRF: output is list[list[int]], pred_ids from output[0], confidence = 1.0 for all spans
       - Use decode_bio_to_char_spans from metrics.py for BIO-to-span reconstruction. Then enrich each span with text=text[start:end] and confidence=mean softmax prob of tokens in that span.
       - `predict_batch(self, texts: list[str]) -> list[list[PredictedSpan]]` — loop over texts calling predict(). (Batch tokenization with padding is possible but loop is simpler and sufficient for PoC.)
       - `find_latest_checkpoint(cls, base_dir: Path = None) -> Path` — classmethod that finds the most recent .pt file in CHECKPOINT_BASE by modification time. Raise FileNotFoundError if none found.

    **Important details:**
    - Use `from src.data.bio_converter import LABEL_O, LABEL_B_REF, LABEL_I_REF` for constants
    - Use `from src.evaluation.metrics import decode_bio_to_char_spans` for span reconstruction
    - Use `from src.model.trainer import load_checkpoint, CHECKPOINT_BASE` for checkpoint loading
    - Use `from src.model.ner_model import RegulatoryNERModel` for model creation
    - Use `from transformers import BertTokenizerFast` (not AutoTokenizer — see STATE.md decision)
    - For confidence calculation in non-CRF mode: for each span (start_tok_idx..end_tok_idx), take softmax(logits, dim=-1) at those positions for their predicted class, then average. Requires tracking token indices alongside character offsets during decode. Extend decode logic to also return token index ranges, or compute inline.

    - TestConfidenceScores: verify confidence in [0, 1] for non-CRF, confidence == 1.0 for CRF.
    - TestBatchPredict: verify predict_batch returns list of lists, length matches input.
    - TestNoReferences: model returns all-O predictions, verify empty list returned.
    - TestFindLatestCheckpoint: create temp dir with .pt files, verify correct one found.

    Mock pattern: patch `RegulatoryNERModel` and `load_checkpoint` and `BertTokenizerFast.from_pretrained` so no real model/tokenizer is loaded. Return a mock model whose forward() returns a mock with .logits attribute (non-CRF) or list[list[int]] (CRF).
