    2. `generate_gold_set(config, num_samples=50, output_path=None) -> list[dict]`:
       - Uses fixed seed from `config.data.llm_seed`
       - For each sample i in range(num_samples):
         - Determine if positive or negative: `include_refs = (i / num_samples) >= config.data.negative_sample_ratio` (first N are negative, rest positive -- deterministic split)
         - Actually: use seed-based deterministic selection. For sample i, if `i < num_samples * negative_sample_ratio` then negative, else positive. This ensures reproducible ratio.
         - Seed for this sample: `config.data.llm_seed + i`
         - Domain: `get_domain_for_seed(config.data.llm_seed + i)`
         - Call LLM (async via asyncio.run or gather for batch), parse ref tags
         - Convert to BIO via char_spans_to_bio
         - Build sample dict: `{"text": clean_text, "spans": spans, "bio_labels": labels, "needs_review": True, "domain": domain, "seed": seed, "has_references": include_refs}`
       - Returns list of sample dicts

    3. `save_gold_set(samples, output_path)`: Writes `json.dumps(samples, ensure_ascii=False, indent=2)` to output_path.

    4. `main()`: Loads config via `load_config()`, determines output path from `config.data.gold_test_dir / "gold_test_set.json"`, calls `generate_gold_set` then `save_gold_set`. Prints summary: N total samples, N positive, N negative.

    5. Script entry: `if __name__ == "__main__": main()`
