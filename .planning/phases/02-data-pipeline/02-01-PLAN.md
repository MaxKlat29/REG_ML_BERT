  <action>
    Create `src/data/llm_client.py` implementing:

    1. **DOMAIN_LIST**: List of 13 German regulatory domain abbreviations.
    2. **RetryableAPIError(Exception)**: Custom exception for retryable HTTP status codes.
    3. **RETRYABLE_STATUS = {429, 408, 502, 503}**: Set of status codes that trigger retry.
    4. **call_openrouter(client, model, messages, seed, api_key, max_retries=5)**: Async function decorated with `@retry(stop=stop_after_attempt(max_retries), wait=wait_exponential_jitter(initial=1, max=60, jitter=5), retry=retry_if_exception_type(RetryableAPIError))`. Posts to `https://openrouter.ai/api/v1/chat/completions` with JSON body `{"model": model, "messages": messages, "seed": seed}` and `Authorization: Bearer {api_key}` header. Timeout 30s. Raises RetryableAPIError on retryable status. Returns `response.json()["choices"][0]["message"]["content"]`.

    NOTE on retry decorator: The `max_retries` parameter cannot be passed dynamically to a decorator. Use a fixed `stop_after_attempt(5)` in the decorator. For test speedup, tests should patch `tenacity.wait.wait_exponential_jitter` to `wait_none()` or patch the retry object's wait attribute.

    5. **parse_ref_tags(tagged_text) -> tuple[str, list[tuple[int, int]]]**: Uses `re.compile(r"<ref>(.*?)</ref>", re.DOTALL)` to find all ref tags. Iterates matches, tracks character offset adjustment as tags are removed, builds clean text and span list. Validates each span: `assert clean_text[start:end]` is non-empty and matches the ref content.

    6. **build_generation_prompt(domain, include_references=True) -> str**: Returns German prompt string requesting regulatory text from the given domain. If include_references=True, asks for `<ref>` tagged references with example. If False, asks for text with NO references.

    7. **get_domain_for_seed(seed) -> str**: Returns `DOMAIN_LIST[seed % len(DOMAIN_LIST)]`.

    Use stdlib `re` for ref tag parsing (simple non-nested case; the `regex` PyPI package is for the complex baseline patterns). Use `import os` to read `OPENROUTER_API_KEY` from env as fallback if not passed.
